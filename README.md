# Earthquake_EcoGeomorphic-work

本项目旨在通过长时序遥感分析与高精度地形数据的深度融合，量化地震扰动景观下的生态与地貌协同恢复机制，揭示由同震滑坡诱发并长期存在的“地震粗糙度记忆”对植被恢复与边坡稳定的控制作用。整体研究思路与方法框架分为以下四个核心逻辑步骤：

###  第一部分：长时序植被突变检测（CCDC时间序列分割）
本步骤依托 **Google Earth Engine (GEE)** 平台，构建了跨越三十年的 Landsat 密集时间序列。核心思路是采用**连续变化检测与分类（CCDC）简谐模型**对植被指数（NDVI）的动态轨迹进行建模。通过设定严格的卡方检验概率与时间节点阈值，精准捕捉同震期间植被覆盖的显著断点（Break），从而大范围提取由地震直接诱发的植被破坏区（VDA）与潜在滑坡活动信号。

###  第二部分：地貌形态学感知的滑坡提取（GAM算法提取与修正）
针对像素级时间序列检测易产生假阳性及边界破碎的问题，本步骤引入了 **GAM（Geomorphologically-Aware Morphology）算法**对突变像元进行物理机制约束。整体提取思路分为三个阶段：
1. **源区种子点识别**：提取具备极显著变化且位于陡坡区域的高置信度像元作为滑坡源区种子点；
2. **水文连通性生长**：结合高精度 DEM 提取的 D8 水文流向，驱使种子点向下游连通生长，聚合滑动路径像素直至平缓沉积区；
3. **空间形态学修正**：最后，通过形态学闭运算填补多边形内部空洞并平滑边界，从而物理真实地还原滑坡体的空间形态。

###  第三部分：地形差分与地表特征粗糙度提取
本步骤从二维平面的生态破坏延伸至三维空间的地貌重塑。利用震前震后高分辨率 DEM 数据进行**高程差分运算**，量化同震地表物质的深度运移量。在此基础上，通过局部窗口标准差计算滑坡表面的**特征粗糙度**。该步骤的目的是从空间上提取地震刻画在地形上的物理印记，论证约 10 米量级的特征高程变化是如何作为一种“地貌记忆”，在震后数十年内持续改变局部水文与土壤条件的过程。

###  第四部分：植被恢复轨迹建模与生态-地貌耦合分析
在明确滑坡边界与地表粗糙度属性后，本步骤对滑坡面上的 NDVI 恢复轨迹进行长期追踪，并应用**逻辑斯蒂（Logistic）生长模型**对恢复曲线进行非线性拟合。该模型旨在量化不同地貌背景下的**植被恢复率（VRR）**及达到特定恢复阈值（如震前水平的 80%）所需的时间。通过将恢复时间与特征粗糙度进行耦合，最终揭示同震物质运移尺度对后续几十年生态演替轨迹的非线性抑制效应。
